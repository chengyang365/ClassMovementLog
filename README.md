新廊华小电子出入记录系统 (SJKCLG Class Movement Log System)这是一个基于 React, Vite 和 Tailwind CSS 开发的现代化班级出入记录系统。旨在取代传统的纸质记录簿，提供更高效、环保且数据可视化的校园管理方案。📋 目录项目简介功能特性技术栈快速开始环境准备安装步骤项目结构使用指南部署📖 项目简介该系统专为小学课堂场景设计（特别是平板电脑/触屏设备）。它允许学生自助登记离开和返回班级的时间，同时为老师提供实时的监控面板和自动化的统计报表。系统支持中马双语，界面直观，不仅能提高纪律管理的效率，还能通过数据分析发现潜在的学生行为问题。✨ 功能特性核心功能学生自助出入登记：大按钮设计，适合触屏操作。学生选择名字 -> 选择原因（如厕所、办公室、图书馆等）即可完成登记。实时监控仪表盘：显示当前日期、时间和班级信息。“正在外面”列表：实时显示谁不在班上。超时预警：学生外出超过 15 分钟，名字会自动变红闪烁，提醒老师注意。学生归队：点击“正在外面”的学生卡片即可标记回来，系统自动计算外出时长。数据与管理统计分析 (Statistics)：月度/年度总览：柱状图显示出入频次。原因分布：饼图分析学生外出的主要原因。时段分析：显示哪个时间段（如 9:00 AM）外出最频繁。高频名单：自动统计“外出王”和“超时王”。历史记录 (History)：查看详细的出入流水账，支持导出 CSV (兼容 Excel) 和打印报表。每日点名 (Attendance)：老师可标记缺席学生，缺席者将无法操作外出功能。系统设置多设备绑定：一台设备可绑定多个班级（如 1A 和 1B）。考试模式 (Exam Mode)：管理员开启后，锁定外出功能，防止干扰考试。数据备份与恢复：支持导出 JSON 格式的全量数据备份，防止数据丢失。离线优先：数据存储在本地 (LocalStorage)，无网络环境下也能正常运行核心功能。🛠 技术栈前端框架: React 18构建工具: Vite样式库: Tailwind CSS图标库: Lucide React图表库: Chart.js🚀 快速开始环境准备确保您的电脑已安装 Node.js (推荐 v16 或更高版本)。安装步骤克隆或下载代码git clone <your-repo-url>
